# **App Name**: ReelGenius

## Core Features:

- Viral Hook Generator: Generate attention-grabbing hooks for reels, tailored for different platforms.
- Reel Script Generator: Generate scripts for 15s, 30s, and 60s reels in Hinglish, Hindi, and English, with storytelling structure.
- Caption and Hashtag AI: Generate emotion-based captions and trending hashtags.
- Call-to-Action Generator: Generate compelling calls to action such as follow, save, comment, and sales prompts.
- Multi-Style Rewrite Tool: Rewrite generated content in various styles, including aggressive, calm, emotional, storytelling, and authoritative tones, using an LLM tool.
- Language Switching: Enable one-click switching between Hindi, Hinglish, and English.

## Style Guidelines:

- Primary color: Vibrant Purple (#9D4EDD) to convey creativity and modernity, fitting for content generation tools.
- Background color: Light Gray (#F5F5F5), desaturated, from the same purple hue family for a clean backdrop.
- Accent color: Electric Blue (#48BFE3), analogous to purple but distinct in brightness and saturation for interactive elements.
- Font: 'Inter', a grotesque-style sans-serif, to be used for headlines and body text.
- Use clear, modern icons to represent features and actions. Favor a consistent style (filled or outlined).
- Maintain a clean, intuitive layout with clear sections for input, generation, and output.
- Use subtle animations for generating new content to provide user feedback.
